Download

Prerequisities

Java SDK 1.8, Maven 3.1.0, (Git - Optional)

Code

Heiko is hosted on GitHub. It can be cloned by GIT or downloaded as a ZIP file. 


# development version as ZIP file
curl https://github.com/gabert/sdx-controller/archive/devel.zip

# development version GIT clone
git clone https://github.com/gabert/sdx-controller.git
cd sdx-controller
git checkout devel

# release version
curl https://github.com/gabert/sdx-controller/archive/release.zip

# release version GIT clone
git clone https://github.com/gabert/sdx-controller.git
cd sdx-controller
git checkout release
