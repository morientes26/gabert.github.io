Heiko Sample Application: Run

Checkout or download release branch as described in download section. After downloading or cloning project, change working directory to project root. Then enter: 


mvn clean install
cd sample-app
mvn exec:java -Dex=1
After a while browser window will open showing Data Tree browser.

Right side shows our managed components organized in tree structure. For a while just ignore system node. We'll get back to it in developers guide section. By unpacking individual nodes we get to clima and/or thermometer property leaves. Next to leaves there are three buttons for each property leaf. By clicking GET button, property value will be retrieved from managed component. Accordingly SET sets value of property. By clicking on LISTEN button, application will register value change listener. By setting LISTENER we will be notified, when value of property leaf changed. All retrieved values from GET or as a result of LISTENER are shown at the left side of an application window.

To demonstrate controlling functionality of Heiko, peform following steps.

  • Check state of Clima power status by clicking GET button. Clima shows OFF state. Thermometer/temperature shows value 19
  • To demonstrate that our logic deployed in controller is working properly, register listener to clima/power property leaf by clicking LISTEN button.
  • SET temperature value to 22. Immediately we'll be notified about clima/power value change to MEDIUM in Console on the left. We can also retrieve value from clima/power by clicking GET next to clima/power.
